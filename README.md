## USB

-----|------|---------|------
TP1  |GND   |Gnd      |Black
TP2  |USB DM|Data-    |White
TP3  |USB DP|Data+    |Green
PIN39|VSYS  |5V       |Red
PIN40|VBUS  |5VD
